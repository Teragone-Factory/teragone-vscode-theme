# Screenshots

The main `README.md` references three PNGs in this directory:

- `editor.png` — a code editor view showing syntax highlighting across a
  representative mix of languages (Python, TypeScript, Markdown).
- `markdown-preview.png` — the VSCode markdown preview rendered with the
  bundled `styles/teragone-markdown-preview.css`.
- `mermaid.png` — a mermaid diagram in the markdown preview, showing the
  re-skinned nodes and edges.

These files are tracked as placeholders until real screenshots are
captured (see `BACKLOG.md` → "Capture real screenshots"). Until then the
README image references will 404 on GitHub; this is expected.

## Capture hints

- Use a 1440×900 window size for consistency.
- Pick a representative snippet for each language so the screenshot
  exercises strings, numbers, keywords, function names, comments.
- Export PNG at 2x for retina clarity, then compress with `pngquant` or
  `oxipng`.
